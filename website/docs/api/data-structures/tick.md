# Tick

## Tick
Checks if a symbol with a specified name exists.

### Example
```
exist = SymbolExist()
```

### Props
|  Name       |  Type           |  Default  |  Description   |
| ----        | ----            | ----      | ----           |
| name        | str             | ----      | Symbol name    |

### Return
If false, the symbol is not found.

|  Name       |  Type    |  Default  |  Description   |
| ----        | ----     | ----      | ----          |
| balance      | float      |           |  If false, the symbol is not found.  |
